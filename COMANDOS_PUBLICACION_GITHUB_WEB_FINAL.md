# Comandos de publicación · Web candidata final

Publicación prevista desde Termux o PC, sustituyendo la raíz del repositorio tras crear una rama de copia de seguridad.

Pasos mínimos:

1. Verificar SHA-256 del ZIP descargado.
2. Extraer el paquete.
3. Crear rama `backup-antes-web-final-*`.
4. Sustituir la raíz del repositorio por el contenido del paquete.
5. Ejecutar verificaciones.
6. Hacer commit y push a `main`.

Comandos exactos: usar el bloque entregado junto al ZIP final.
