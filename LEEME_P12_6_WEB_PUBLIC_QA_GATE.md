# LEEME · P12-6 Web public QA gate

Esta fase prepara una carpeta estática `dist/github_pages/` para publicar la app en GitHub Pages.

## Qué probar

1. Subir el contenido de `dist/github_pages/` al repositorio o rama de Pages.
2. Abrir la URL pública.
3. Cargar `Ejemplo web P12-5`.
4. Generar horario.
5. Confirmar que el motor usado es `WEB_SOLVER`.
6. Confirmar que no se pide Python, OR-Tools ni servidor local.

## Limitación

P12-6 no acredita Frián real completo. Es un gate de publicación web y QA, no una ampliación de restricciones.
