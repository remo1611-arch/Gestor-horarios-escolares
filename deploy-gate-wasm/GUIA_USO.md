# Guía de uso · v1.4.1

## Flujo recomendado

1. Abrir la app en local o GitHub Pages.
2. Crear un proyecto nuevo o cargar un ejemplo sintético.
3. Revisar datos del centro, días y tramos.
4. Configurar profesorado, grupos, espacios y actividades.
5. Usar el asistente para detectar bloqueos previos.
6. Calcular horario.
7. Revisar validación, bloqueos, carga docente y calidad.
8. Editar manualmente con **Mover aquí** cuando sea necesario.
9. Exportar JSON e informes.
10. Usar **Gestión diaria** para ausencias/coberturas sin alterar el horario semanal base.

## Tramos y guardias

Las guardias no son tramos. Son actividades o servicios de docencia complementaria colocados dentro de tramos como entrada, recreo o salida.

## Importación CSV

Orden recomendado:

1. Docentes.
2. Grupos/subgrupos.
3. Espacios/zonas.
4. Tramos.
5. Actividades.

Las actividades se importan al final porque necesitan que ya existan los nombres de docentes, grupos y espacios.

## Edición táctil

En móvil o tableta se recomienda toque corto sobre una sesión y después **Mover aquí**. El arrastre queda reservado para dispositivos donde sea fiable.
