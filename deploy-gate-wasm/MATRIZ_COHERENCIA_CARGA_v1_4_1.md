# Matriz de coherencia de carga · v1.4.1

| Aspecto | Estado | Observación |
|---|---|---|
| DC como categoría principal | Soportado | Actividades de docencia complementaria se cuentan separadas de lectivas. |
| Servicios como subconjunto | Soportado | Guardias de entrada, salida, recreo y patio cuentan como servicios aunque sean DC. |
| Objetivo / definido / colocado | Soportado | Diagnóstico visible en Calcular horario e informe TXT. |
| Avisos de carga reales | Soportado | No se silencian ni se reducen artificialmente. |
| Ajuste automático de objetivos | No soportado | Debe ser decisión humana. |
| Garantía de jornada real completa | No soportado | Depende de que las actividades introducidas representen la jornada real. |
| Prueba con datos reales | Pendiente | Debe hacerse con datos reales o anonimizados. |
