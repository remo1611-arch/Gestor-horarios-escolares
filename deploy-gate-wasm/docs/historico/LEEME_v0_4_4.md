# LEEME v0.4.4

## Dictamen

Versión de consolidación sobre v0.4.3. No cambia el motor, el validador, el formato JSON, las condiciones docentes ni las exportaciones.

## Cambios

- Añade botón discreto **Pantalla completa** en la cabecera.
- El botón se oculta si el navegador no soporta la API de pantalla completa.
- Actualiza el texto del botón al entrar/salir de pantalla completa.
- Mantiene el estado accesible `aria-pressed`.
- No marca cambios pendientes del proyecto.
- Mantiene los seguros de v0.4.3: selección cancelable, confirmación al cerrar/recargar con cambios y copia de emergencia.

## Prueba manual mínima

1. Cargar ejemplo complejo.
2. Calcular horario.
3. Entrar en **Revisar y ajustar**.
4. Activar pantalla completa desde la cabecera.
5. Mover una sesión con **Mover aquí**.
6. Salir de pantalla completa.
7. Guardar/exportar JSON.

## Estado

Base recomendada antes de continuar con nuevas funciones.
