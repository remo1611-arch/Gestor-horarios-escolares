# LEEME · v1.3.1

## Naturaleza de la versión

`v1.3.1` es una fase de limpieza, consolidación y auditoría interna sobre `v1.3.0`. No añade funciones nuevas de horario. Su objetivo es dejar la app más clara, coherente y verificable antes del QA físico.

## Cambios realizados

- Documentación histórica movida a `docs/historico/`.
- Documentación activa renombrada y consolidada.
- README actualizado a la versión real.
- Versión de app consolidada a `1.3.1`.
- Formato JSON mantenido en `horario-escolar-json-1.7`.
- Ejemplos sintéticos actualizados a `1.3.1`.
- Nuevas pruebas de coherencia documental y versión.
- Regeneración de manifiesto SHA-256.

## Qué no cambia

- Motor incremental.
- Modelo de tramos.
- Importación CSV.
- Calidad del horario.
- Gestión diaria.
- Estructura del formato JSON.

## Estado

Candidata funcional consolidada. Pendiente de QA físico, GitHub Pages real, impresión A4/PDF y prueba privada con datos reales o anonimizados.
