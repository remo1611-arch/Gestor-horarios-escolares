## Criterio de aceptación

La versión 0.3.0 se considera consolidada si:

- el cálculo puede cancelarse sin alterar el horario anterior;
- el guardado informa correctamente éxito o fallo;
- se puede recuperar copia actual y anterior;
- las operaciones destructivas permiten cancelar;
- no aparecen errores graves tras movimientos válidos;
- las exportaciones básicas siguen funcionando.
