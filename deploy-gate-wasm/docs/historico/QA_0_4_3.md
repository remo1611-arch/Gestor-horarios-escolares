## Controles añadidos

- Existe función única de cancelación de selección temporal.
- Existe botón Cancelar selección.
- Cambiar de sección limpia selección activa.
- Cambiar vista o entidad de revisión limpia selección activa.
- Tocar zona no interactiva del editor cancela selección.
- `beforeunload` guarda emergencia y activa confirmación nativa si hay cambios pendientes.
- `visibilitychange` y `pagehide` guardan copia de emergencia.
- Recuperar copia anterior/emergencia pide confirmación explícita.
- Recalcular sobre horario existente pide confirmación.
