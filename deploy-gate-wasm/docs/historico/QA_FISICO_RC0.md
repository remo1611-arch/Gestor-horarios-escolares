# QA físico · RC0

## Perfiles obligatorios antes de v1.0 estable

1. Android móvil.
2. Tableta horizontal.
3. PC Windows o Linux con Chrome/Edge.
4. GitHub Pages real.
5. Impresión A4 / Guardar como PDF.

## Recorrido común

Para cada perfil:

1. Abrir la app.
2. Cargar `Centro exigente sintético`.
3. Revisar tramos reales.
4. Calcular horario.
5. Revisar explicación de bloqueos.
6. Revisar carga docente.
7. Mover una sesión con el editor.
8. Probar cancelar selección.
9. Probar pantalla completa si procede.
10. Exportar JSON.
11. Exportar informe de validación.
12. Exportar informe de bloqueos.
13. Exportar informe de carga.
14. Abrir HTML imprimible.
15. Guardar como PDF o imprimir A4.

## Resultado esperado

- Sin pantalla en blanco.
- Sin pérdida de proyecto al editar.
- Sin bloqueo táctil por pulsación prolongada.
- Sin sesiones inválidas aceptadas como válidas.
- Informes descargables legibles.
- JSON exportado reimportable.

## Evidencia mínima por perfil

- Dispositivo/navegador.
- URL o modo local.
- Fecha.
- Resultado del recorrido.
- Incidencias observadas.
- JSON exportado de prueba si procede.
