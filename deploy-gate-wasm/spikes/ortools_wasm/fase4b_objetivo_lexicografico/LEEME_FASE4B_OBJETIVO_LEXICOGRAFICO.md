# LEEME · Fase 4B CP-SAT lexicográfico

Esta carpeta contiene una prueba experimental de OR-Tools CP-SAT WebAssembly con objetivo lexicográfico.

No modifica la app funcional ni sustituye el motor actual.

## Pruebas

```bash
node spikes/ortools_wasm/fase4b_objetivo_lexicografico/tests/fase4b_lexicografico_node.mjs
node spikes/ortools_wasm/fase4b_objetivo_lexicografico/tests/fase4b_validacion_cruzada_ejemplos.mjs
```

La validación completa se puede regenerar con:

```bash
node spikes/ortools_wasm/fase4b_objetivo_lexicografico/tests/fase4b_validacion_cruzada_ejemplos_completa.mjs
```
