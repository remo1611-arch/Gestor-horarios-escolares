# P12-5 · Publicación web estática / GitHub Pages

P12-5 prepara una distribución estática para abrir la aplicación desde un enlace y trabajar localmente en el navegador.

## Lo que queda permitido

- Abrir desde GitHub Pages o servidor estático equivalente.
- Cargar ejemplos públicos.
- Generar con `WEB_SOLVER` los casos compatibles P12-1/P12-2.
- Guardar en el navegador y exportar archivos locales.
- Usar CP-SAT únicamente fuera de la web pública como oráculo técnico de desarrollo.

## Lo que queda prohibido o no acreditado

- Subir proyectos reales a un backend remoto.
- Declarar que Frián real genera completo en navegador.
- Usar CP-SAT como dependencia del usuario final en GitHub Pages.
- Mezclar esta fase con XADE, documentos, XLSX o cambios de interfaz no necesarios.

## Alcance acreditado

- `P12_WEB_MINI`: generación web simple.
- `P12_ORG41_LIGHT`: generación web organizativa ligera.

## Pendiente

- Despliegue real en GitHub Pages.
- QA físico desde enlace público.
- Motor web medio para casos más parecidos a centros reales.
- Paridad real contra CP-SAT ejecutada en Windows/OR-Tools.
